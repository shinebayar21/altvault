# 🛍️ Онлайн дэлгүүр

Next.js + TypeScript + SQLite дээр хийсэн, бүрэн үнэгүй технологиор ажилладаг онлайн дэлгүүр.

## Ажиллуулах

```bash
npm install
npm run dev
```

Дараа нь хөтчөөрөө http://localhost:3000 руу орно.

Production горимоор ажиллуулах бол:

```bash
npm run build
npm start
```

## Хэсгүүд

| Хаяг | Тайлбар |
|---|---|
| `/` | Дэлгүүрийн нүүр — бараа, категори, хайлт |
| `/products/[id]` | Барааны дэлгэрэнгүй |
| `/cart` | Сагс |
| `/checkout` | Захиалга өгөх |
| `/order/[код]` | Захиалга шалгах, төлбөрийн заавар |
| `/admin` | Админ хэсэг (нууц үг: `.env.local` доторх `ADMIN_PASSWORD`, анхны утга `admin123`) |

## Админ хэсэг

- 📊 Хянах самбар — орлого, захиалгын тоо, бага үлдэгдэлтэй бараа
- 📦 Захиалга — төлбөр баталгаажуулах, хүргэгдсэн болгох, цуцлах (цуцлахад үлдэгдэл буцна)
- 🏷️ Бараа — нэмэх/засах/хаах, зураг upload, категори нэмэх
- 🔧 Тохиргоо — дэлгүүрийн нэр, утас, банкны данс

## Төлбөр

**Дансаар:** Захиалга өгмөгц хэрэглэгчид данс + гүйлгээний утга (захиалгын код) харагдана.
Мөнгө орж ирсний дараа админ хэсгээс "Төлбөр баталгаажуулах" дарна.

**QPay:** Кодонд бүрэн холболт бэлэн. QPay-тэй мерчант гэрээ хийгээд `.env.local` дотор
`QPAY_USERNAME`, `QPAY_PASSWORD`, `QPAY_INVOICE_CODE`-г бөглөөд серверээ дахин асаахад
захиалга бүрд QR код автоматаар үүсч, төлбөр автоматаар баталгаажина. Гэрээгүй үед
QPay сонгосон захиалга дансаар төлөх горим руу аяндаа шилждэг.

## Өгөгдлийн сан

SQLite — `data/shop.db` файлд хадгалагдана, сервер/бүртгэл шаардлагагүй, бүрэн үнэгүй.
Анх ажиллуулахад жишээ бараанууд автоматаар үүснэ. Барааны зургууд `data/uploads/` дотор хадгалагдана.
**Анхаар:** `data/` фолдерыг үе үе хуулбарлаж нөөцлөөрэй.

## Аюулгүй байдал

`.env.local` доторх `ADMIN_PASSWORD` болон `AUTH_SECRET`-ийг заавал солиорой.
